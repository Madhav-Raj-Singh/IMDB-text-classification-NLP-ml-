# 🎬 IMDb Sentiment Analysis (NLP)

## 📌 Overview
This project performs Sentiment Analysis on the IMDb movie review dataset.  
The goal is to classify reviews as Positive or Negative using NLP techniques and Machine Learning models.

---

## ⚙️ Pipeline
1. Text Cleaning & Preprocessing  
2. Tokenization & Stopword Removal  
3. Word2Vec Embedding  
4. Average Word Vectors → Document Vector  
5. Train-Test Split  
6. Feature Scaling  
7. Model Training & Evaluation  

---

## 🤖 Models Used
- Logistic Regression  
- Random Forest Classifier  

---

## 📊 Evaluation
- Metric: Accuracy Score  
- Logistic Regression performed better than Random Forest on Word2Vec features.
